<Tip>Category: **Attack Surface Management**</Tip>

<Note>SSL certificates obtained from various hosts may contain valuable information that could be utilized to create a wordlist and then find additional hostnames belonging to a specific company's root domain.</Note>

<Check>Complexity: Basic</Check>

## Setup

You can set up this workflow by changing the following input value:

* **TARGET** - Provide a file containing the company's CIDR or IP-range, as a target.

### Execution and Results

After setup, the workflow is ready to be executed. Once the workflow's last node, `recursively-cat-all` script, is finished, the result can be viewed and downloaded.

The [recursively-cat-all](https://trickest.io/dashboard/library/script/f00bd5d5-8b14-49aa-80ac-d70dbff5dcaf) script will contain the wordlists gathered from `cero` parallel execution for each CIDR list.
